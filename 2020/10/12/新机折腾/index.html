Linux基本环境配置

Posted on 2020-10-12

Ucloud云主机

贪便宜买一了台ucloud服务器，1核2G 2M带宽，系统盘40G

一、服务器配置调整

  • 修改服务器名字
  • 安装rz、sz
  • 新增用户

Ps:
1、默认会在/home下创建一个用户同名的文件夹
2、点击查看相关命令

  • 普通用户增加sudo权限

Ps:
/etc/sudoers中#includedir /etc/sudoers.d前面的#不是注释的意思，不能去掉

  • 关闭ssh默认的root登录
  • 修改ssh登录端口

Ps:
用的ucloud，需要在网络基础=>外网防火墙 中创建防火墙规则，或在已有规则下添加新开放端口，否则公网无法访问
如果是新建规则，需要在云主机=>...=>关联产品操作=>修改外网防火墙中选择新建的规则

二、安装php环境

  • nginx

Ps:
1、根据自己的需求调整/etc/nginx/nginx.conf中配置
2、点击查看更多yum命令

  • php

php -r “copy(‘https://install.phpcomposer.com/installer', ‘composer-setup.php’);” 下载安装脚本 － composer-setup.php － 到当前目录。
php composer-setup.php 执行安装过程。
php -r “unlink(‘composer-setup.php’);” 删除安装脚本。
mv composer.phar /usr/local/bin/composer 全局安装

wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.36.0/install.sh | bash

npm install cnpm -g –registry=https://registry.npm.taobao.org

yum install git (我是系统默认安装了)

ssh-keygen -t rsa -C ”你的邮箱“ // 也可以不填邮箱

git clone “项目地址”
`

Ps:
1、域名不要直接指向git目录，有暴露git信息的风险
2、前端vue，后端laravel，项目打包更新都是通过shell脚本执行的
